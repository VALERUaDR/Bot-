Акторы:
    Пользователь
    Бот
    Совпадение

- Пользователь открывает диалог с ботом, пишем ему приветсвтие с командой активации бота
- Пользователь активирует бота командой /start
    1. Проверить данные пользователя
    1.1 Если нет возраста спросить пользователя
    1.2 Если нет пола спросить пользователя
    1.3 Если нет города спросить пользователя
    2. Сохранить данные пользователя в БД
- Пользователь отправляет команду поиска совпадений
    Бот ищет по данным пользователя совпадения, и из полученного списка берёт случайного человека,
     если ещё не было промотра и профиль не закрыт, берем 3топ фото, отмечаем пользователя просмотренным
     Отправляем пользователю ссылку и фотки

Структура БД:
